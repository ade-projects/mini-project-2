# Mini project 2 : Analisis Kelulusan dan Predikat Wisuda Mahasiswa

Proyek ini adalah program Python sederhana untuk menganalisis data wisudawan dari file Excel. Program ini memproses data untuk mengklasifikasikan nilai (IPK) ke dalam grade (A, B+, B, C, D) dan menentukan predikat kelulusan (Cumlaude, Sangat Memuaskan, Memuaskan, Cukup) berdasarkan kriteria tertentu. Selanjutnya hasil analisis divisualisasikan menggunakan diagram untuk mempermudah pemahaman data.
